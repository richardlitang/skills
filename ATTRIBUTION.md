# Attribution

This repository packages locally maintained Codex skills from `~/.codex/skills`.

## Included Sources Of Inspiration

| Area | Inspiration |
| --- | --- |
| Codex skill structure | Local Codex skill conventions: `SKILL.md` frontmatter, progressive disclosure, optional `references/`, `scripts/`, and `assets/` folders. |
| Engineering workflow skills | Repeated personal-project agent workflows for planning, debugging, verification, code review, and duplicate-function audits. |
| Android skills | Android, Gradle, Jetpack Compose, Material 3, accessibility, localization, and test-driven development practices. |
| SEO skills | Google Search Central, Schema.org, Core Web Vitals guidance, technical SEO practice, content quality review patterns, and AI-search visibility workflows. |
| UX writing skill | Product microcopy practice, accessibility-oriented UX writing, error-message patterns, empty states, onboarding copy, and voice-and-tone frameworks. |
| Interface and frontend design skills | Product interface design practice, design critique workflows, accessibility checks, responsive UI validation, and frontend implementation patterns. |
| PumpingChops skills | Personal PumpingChops content QA and editorial workflows for review, roundup, and informational pages. |

## Exclusions

System-provided Codex skills were not copied into this repo.

Buildpass skills, Buildpass prompts, Buildpass workflows, and anything derived from Buildpass are intentionally excluded. The validator fails if Buildpass references are introduced under `codex/`.
