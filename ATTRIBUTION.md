# Attribution

This repository packages locally maintained Codex skills from `~/.codex/skills`.

## Included Sources Of Inspiration

| Area | Inspiration |
| --- | --- |
| Agent skill methodology | [obra/superpowers](https://github.com/obra/superpowers), especially its framing of composable agent workflows for brainstorming, planning, TDD, review, and verification. |
| Codex skill structure | Local Codex skill conventions and the public [OpenAI skills repository](https://github.com/openai/skills), especially `SKILL.md` frontmatter, progressive disclosure, optional `references/`, `scripts/`, and `assets/` folders. |
| Skill authoring model | [Anthropic Skills overview](https://claude.com/docs/skills/overview), [Anthropic Agent Skills docs](https://docs.claude.com/en/docs/agents-and-tools/agent-skills), and [Skill authoring best practices](https://docs.claude.com/en/docs/agents-and-tools/agent-skills/best-practices), especially progressive disclosure and resource loading. |
| Engineering workflow skills | Repeated personal-project agent workflows for planning, debugging, verification, code review, and duplicate-function audits, with conceptual overlap from Superpowers' workflow-first skill library. |
| Android skills | [Android Developers Jetpack Compose documentation](https://developer.android.com/develop/ui/compose/documentation), [Compose accessibility guidance](https://developer.android.com/develop/ui/compose/accessibility), [Compose semantics](https://developer.android.com/jetpack/compose/semantics), and [Material 3 in Compose](https://developer.android.com/develop/ui/compose/designsystems/material3). |
| SEO skills | [Google Search Central SEO Starter Guide](https://developers.google.com/search/docs/fundamentals/seo-starter-guide), [Google Search reviews system](https://developers.google.com/search/docs/appearance/reviews-system), [Google outbound link qualification](https://developers.google.com/search/docs/crawling-indexing/qualify-outbound-links), [Schema.org](https://schema.org/), and [web.dev Web Vitals](https://web.dev/articles/vitals). |
| WordPress content skills | Personal WordPress content QA and editorial workflows, informed by the [WordPress Theme Handbook](https://developer.wordpress.org/themes/), [WordPress Plugin Handbook](https://developer.wordpress.org/plugins/), and [WordPress template hierarchy](https://developer.wordpress.org/themes/templates/template-hierarchy/). |
| UX writing skill | Product microcopy practice, accessibility-oriented UX writing, error-message patterns, empty states, onboarding copy, [Microsoft writing style](https://learn.microsoft.com/en-us/windows/apps/design/style/writing-style), [Microsoft error message guidelines](https://learn.microsoft.com/en-us/windows/win32/debug/error-message-guidelines), and [Material Design writing guidance](https://m1.material.io/style/writing.html). |
| Interface and frontend design skills | Product interface design practice, design critique workflows, accessibility checks, responsive UI validation, [WCAG 2.2 understanding docs](https://w3c.github.io/wcag/understanding/intro), and frontend implementation patterns. |

System-provided Codex skills were not copied into this repo.
